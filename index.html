Project 2

In this project (using Java), the user will read a set of 2D integer points on a coordinate plane. The objective is to determine the median coordinate point, where the x-coordinate is the median of the input points' x-coordinates, and the y-coordinate is the median of their y-coordinates. The input points should be processed four times, each time using one of the following sorting algorithms: selection sort, insertion sort, merge sort, and quicksort.
